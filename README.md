# scale_gds — Geriatric Depression Scale

A 30-item (or 15-item short form) yes/no self-report scale for depression in older adults. Designed to be simple to administer and minimize somatic items that can confound depression ratings in elderly patients.

- **Instrument:** GDS (GDS) · Depression — Geriatric
- **15 items** (see `data/gds.json`).
- **Scoring:** GDS-30: 0–9 normal; 10–19 mild depression; 20–30 severe depression. GDS-15: 0–4 normal; 5–8 mild; 9–11 moderate; 12–15 severe.
